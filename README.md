# Learning Bubbles

一個新的 Flutter 專案。

## 開始使用

### 前置需求

- Flutter SDK (3.0.0 或更高版本)
- Dart SDK
- Android Studio / Xcode (用於開發移動應用)

### 安裝步驟

1. 確保已安裝 Flutter：
   ```bash
   flutter doctor
   ```

2. 安裝依賴：
   ```bash
   flutter pub get
   ```

3. 運行應用：
   ```bash
   flutter run
   ```

## 專案結構

```
learningbubbles/
├── lib/
│   └── main.dart          # 應用程式入口
├── android/               # Android 平台配置
├── ios/                   # iOS 平台配置
├── pubspec.yaml          # 專案依賴配置
└── README.md             # 本文件
```
